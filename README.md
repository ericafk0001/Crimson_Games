## Hosting your own
Make sure to have Node.js and NPM installed
`
git clone https://github.com/ericafk0001/Crimson_Games
`

`
cd https://github.com/ericafk0001/Crimson_Games
`

`npm install`

**Building**

```bash
npx tailwindcss -i ./src/input.css -o ./dist/style.css --watch
```
*OR*
```bash
npm run dev
```
